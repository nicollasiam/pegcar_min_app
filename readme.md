# Pegcar - Nicollas Isabel

## Este é um protótipo, proposto pela Pegcar.

## Configurações

* Roda 2.28.0
* tilt
* erubis

## Instalar o sistema

```bash
git clone
cd

bundle install
```

## Iniciar o servidor

Após clonado o projeto, basta iniciar o servidor para acessar a página estática criada

```bash
rackup
```
